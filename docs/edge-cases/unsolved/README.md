## When to add a page here

Add under **`unsolved/`** when:

- The topic is **strategic** (trade-offs, future config schema sketches), or  
- Work is **not yet implemented** but worth preserving for contributors and agents.

Move or duplicate into **`solved/`** once a fix is stable and user-facing — see [edge cases home](../README.md).

# `locales list`

Lists locale JSON files under `localesDir` with per-file leaf counts and source-identical hints.

```bash
i18nprune locales list
i18nprune locales list --json
```

## JSON mode

Global `--json` emits `kind: locales-list` with per-file rows and aggregate counts.

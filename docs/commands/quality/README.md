# `quality` (`q`)

**Full examples:** [quality examples](../../examples/commands/quality/README.md)

Reports **source-identical** (target string equals source locale at path) and parity-style metrics using config **policies** where implemented. Human output includes source/target locale file rows with leaf counts so it lines up with `locales list`.

```bash
i18nprune quality
i18nprune quality --target ja
```

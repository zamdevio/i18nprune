# `languages` (`langs`)

**Full examples:** [languages examples](../../examples/commands/languages)

Lists supported **translation target** languages (code, English, native name) for **`generate --lang`**.

```bash
i18nprune languages
i18nprune langs --filter ja
i18nprune languages --table
```

**`--json`** prints the catalog as JSON. Does not require a config file.
